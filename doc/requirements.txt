*requirements.txt*       For Vim version 8          Last change: 2019 May 30
*requirements* *vim-requirements*

requirements.txt.vim is a vim plugin that contains

    - filetype detection and syntax highlighting for requirements files

Install it with your favorite plugin manager.

You can add to the patterns detected via *g:requirements#is_requirements_file*
